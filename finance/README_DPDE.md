# 金融偏微分方程模拟器说明文档

## 模型概述

这是一个基于**反应-扩散偏微分方程（Reaction-Diffusion PDE）**的金融市场空间-时间模型，结合了：

1. **扩散过程（Diffusion）**: 模拟资金和流动性的空间传播
2. **反应项（Reaction）**: 模拟买卖双方的交互
3. **分布时滞（Distributed Delay）**: 通过记忆场模拟恐惧的滞后效应
4. **罗宾边界条件（Robin BC）**: 模拟与外部市场的交互

### 核心方程

```
∂S/∂t = D_S * ∇²S + r*S*(1-S/K) - (β*S*B)/(1+k*W)
∂B/∂t = D_B * ∇²B + c*(β*S*B)/(1+k*W) - d*B
∂W/∂t = α*(B - W)
```

其中：
- **S(x,t)**: 卖单存量（流动性），空间-时间场
- **B(x,t)**: 买盘力度（资金），空间-时间场
- **W(x,t)**: 恐惧记忆场（分布时滞变量）
- **D_S, D_B**: 扩散系数
- **k**: 恐惧因子
- **α**: 记忆衰减率（控制滞后程度）

### 关键特性

1. **空间异质性**: 考虑市场的空间维度（不同板块、区域）
2. **边界条件**: 左边界（Robin）模拟外部市场影响，右边界（Neumann）模拟封闭系统
3. **时滞效应**: 恐惧记忆场 W 滞后于实际资金 B，体现市场记忆
4. **外部冲击**: 可以模拟央行放水等外部事件

## 使用方法

### 基本使用

```python
from financial_dpde_simulator import FinancialDPDESimulator

# 创建模拟器
sim = FinancialDPDESimulator(
    nx=100,      # 空间网格点数
    L=10.0,      # 空间域长度
    T=50.0,      # 模拟总时间
    dt=0.01      # 时间步长
)

# 运行模拟
sim.run()

# 绘制结果
sim.plot_results(save_path='results.png')
```

### 自定义参数

```python
sim = FinancialDPDESimulator(nx=100, L=10.0, T=50.0, dt=0.01)

# 修改模型参数
sim.D_S = 0.01   # 流动性扩散系数
sim.D_B = 0.5    # 资金扩散系数
sim.k = 2.0      # 恐惧因子
sim.alpha = 0.5  # 记忆衰减率

# 修改边界条件
sim.h_S = 0.1    # 流动性渗透率
sim.S_ext = 5.0  # 外部流动性
sim.h_B = 2.0    # 资金渗透率
sim.B_ext = 0.0  # 外部资金（可在step中动态改变）

sim.run()
sim.plot_results()
```

## 模型参数说明

### 扩散系数

| 参数 | 含义 | 典型值 | 说明 |
|-----|------|--------|------|
| D_S | 流动性扩散系数 | 0.01 | 卖单扩散慢（挂单通常不动） |
| D_B | 资金扩散系数 | 0.5 | 资金扩散快（热钱流动迅速） |

### 反应项参数

| 参数 | 含义 | 典型值 | 说明 |
|-----|------|--------|------|
| r | 卖单再生率 | 0.5 | 新卖单产生的速度 |
| K | 市场容量 | 10.0 | 卖单的上限 |
| β | 基础成交效率 | 0.8 | 无恐惧时的成交效率 |
| c | 赚钱效应转化率 | 0.6 | 成交转化为买盘的比例 |
| d | 资金消耗率 | 0.1 | 资金自然衰减/撤离率 |

### 关键参数

| 参数 | 含义 | 典型值 | 说明 |
|-----|------|--------|------|
| k | 恐惧因子 | 2.0 | 越大，成交效率越低 |
| α | 记忆衰减率 | 0.5 | 越小，记忆越长，滞后越明显 |

### 边界条件参数

| 参数 | 含义 | 典型值 | 说明 |
|-----|------|--------|------|
| h_S | 流动性渗透率 | 0.1 | 与外部市场的连通性 |
| S_ext | 外部流动性 | 5.0 | 外部市场的平均流动性 |
| h_B | 资金渗透率 | 2.0 | 资金的高渗透性 |
| B_ext | 外部资金 | 0.0 | 可动态改变（模拟放水） |

## 模型应用场景

1. **流动性风险分析**: 预测流动性在空间上的分布和演化
2. **资金流动研究**: 分析资金在不同板块间的扩散
3. **市场冲击传播**: 模拟外部事件对市场的影响
4. **恐惧效应研究**: 分析恐惧记忆对市场的影响
5. **政策效果评估**: 模拟货币政策（如放水）的效果

## 数值方法

- **空间离散化**: 中心差分法（Central Difference）
- **时间积分**: 欧拉法（Euler Method）
- **边界条件**: 幽灵点法（Ghost Point Method）

## 稳定性条件

为确保数值稳定，需要满足：

```
dt < dx² / (2 * max(D_S, D_B))
```

当前设置：`dt=0.01`, `dx≈0.1`, `max(D_S, D_B)=0.5`
- 稳定性条件：`0.01 < 0.01/1.0 = 0.01` ✓ 满足
